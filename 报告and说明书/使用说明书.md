## 1 总体说明

此绘图系统包括命令行界面程序和用户交互界面程序。

命令行界面程序用如下指令运行：

```
python cg_cli.py input_path output_dir
```

其中input_path是输入文件路径，输入文件的指令格式与实验要求中的格式一样，就不赘述了。output_dir是输出文件夹的路径，保存画布得到的位图存在这个文件夹中。

用户交互界面程序用如下指令运行：

```
python cg_gui.py
```
